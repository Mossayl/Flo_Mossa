1. 智能自动分类（Tag 智能推荐）

你在浏览不同网页时，系统可以自动判断当前内容属于哪个分类（产品能力 / 模型工程 / 模型训练…），提供推荐。
上传不同笔记和url时，系统可以自动判断当前内容属于哪个分类（产品能力 / 模型工程 / 模型训练…），提供推荐

例如：
URL 包含 arxiv → 推荐“模型工程 / 模型训练”
URL 包含 zhihu / 博客 → 推荐“成长思考”
URL 包含 product / design → 自动选择推荐“产品能力”
笔记 包含 项目规划/产品方案设计→ 自动选择推荐“产品能力”

价值： 降低记笔记时的分类负担。

2. 智能小标题自动生成

你复制的第一段笔记、标题、H1 信息、网页标题等
→ 自动生成初次小标题。

场景： 你在看大模型文章，复制一段，系统自动：
「LLM inference acceleration 方法总结」

3. 贴心的笔记聚合（按标签归档）

例如今天所有笔记→ 自动放到以标签分类的文件夹中
类似 “每日学习流水账”，你无需手工合并。

5. 快捷键全局支持

例如：
Cmd+Shift+N → 打开悬浮窗
Cmd+Enter → 缓存
Cmd+Shift+Enter → 提交
Cmd+V → 自动识别图片还是文本

5. 智能 OCR（截图 → 自动转文本）
你拖一个图，会自动识别：
图片中的代码
图片中的公式
图片中的标题
并作为一条笔记插入，你可以选择“接受”或“删除”。

价值： 特别适合你读论文、AI 技术博客、PPT 截图。

7. 模板化笔记单元（Note Block Template）

可选择：
思考笔记模板
模型结构记录模板
产品 feature log 模板
阅读笔记模板
降低重复记录的成本。

11. 与 Notion / 飞书文档双向同步（未来增强）

可以自动同步到 Notion 某个页面（API）
或飞书文档（SDK）
